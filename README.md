# Wednesday Dashboard

Real-time dashboard for KOEIGO automation agent.

## Tech Stack
- Next.js 14
- Firebase Firestore
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Firebase Structure
- `dashboard/activity/items` - Activity log
- `dashboard/status` - Current status
- `dashboard/stats` - Statistics
- `dashboard/research/items` - Research items
